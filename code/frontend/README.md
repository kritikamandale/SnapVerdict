# SnapVerdict — Frontend

React + Vite frontend for the SnapVerdict damage claim verification demo.

## Setup

```bash
npm install
npm run dev
```

Opens at **http://localhost:5173/**. Expects the FastAPI backend running on port 8000 (see `../api/`).

## Stack

- React 19 + Vite 6
- Plain CSS (dark-mode friendly)
- No routing, no state management library — single-page demo
